# V-Tax Web Service Boilerplate

Project template/boilerplate for building web service V-Tax Apps


## Tech Stack
- MySQL 5.7
- PHP 5.6


## Installation

```bash
  git clone http://git26.v-tax.id:4000/v-tax-research/vws.git
  cd vws
  cp .env.example .env
```

## Run locally with docker

```bash
    docker compose up -d
```
